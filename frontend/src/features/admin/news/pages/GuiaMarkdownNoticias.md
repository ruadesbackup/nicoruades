# Guía rápida de Markdown para redactar noticias

Puedes usar la siguiente sintaxis en el editor para dar formato al contenido de tus noticias:

## Títulos y subtítulos

# Título principal
## Subtítulo
### Sección

## Negrita y cursiva

**Texto en negrita**
*Texto en cursiva*

## Listas

- Elemento de lista
- Otro elemento

1. Primer punto
2. Segundo punto

## Enlaces

[Texto del enlace](https://ejemplo.com)

## Imágenes

![Texto alternativo](https://url-de-la-imagen.com/imagen.jpg)

## Citas

> Esto es una cita destacada.

## Código

`código en línea`

```
Bloque de código
```

---

**Consejos:**
- Puedes combinar negrita y cursiva: `***muy importante***` → ***muy importante***
- Para saltos de línea, deja una línea en blanco entre párrafos.
- Los cambios se ven en la vista previa del editor.
